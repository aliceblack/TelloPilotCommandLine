# TelloPilotCommandLine
Command line controller for Tello Drone (Edu) 2.0

## Install
```
npm install
```
## Start
```
node index
```

## Keyboard
O enter SDK mode  
P takeoff  
H land  

W front  
A counter clockwise  
S back  
D clockwise  
  
I flip front  
J flip left  
K flip back  
L flip right  

Arrow Up - up  
Arrow Down - down  
Arrow Left - left  
Arrow Right - right
